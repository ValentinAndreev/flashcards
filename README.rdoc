== Менеджер флеш-карточек

* Написан на ruby 2.3.1p112 и Rails 4.2.6

* Использует PostgreSQL 9.4

* Для запуска локально: 

1. git clone git@github.com:ValentinAndreev/flashcards.git  
2. bundle install  
3. rake db:migrate  
4. rails s
